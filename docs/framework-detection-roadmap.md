# Framework Detection Roadmap

Current detectors:
- .NET
- Node.js
- Python
- Flutter

Planned detectors:
- Java / Maven / Gradle
- Rust / Cargo
- Go modules
- React Native
- Android (Gradle)
- Unity / Unreal
- Laravel / Django / FastAPI / ASP.NET
- Angular / Vue / Svelte

Detection should combine root-manifest files, canonical folders, and optional shallow content checks.
